# AWS Cloud Troubleshooting Projects

This repository contains hands-on AWS troubleshooting projects designed for entry-level cloud and support roles.

## Projects
- EC2 Web Application Outage – Multi-layer troubleshooting
- EC2 No Internet Connectivity – VPC routing analysis
- S3 Access Denied – IAM permission debugging
- Linux Service Failure – OS-level troubleshooting

Each project follows a structured root cause analysis approach.
